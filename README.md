# Sistema de Eventos em Java

Projeto desenvolvido em Java para cadastro e notificação de eventos.
Feito como atividade da disciplina de Programação.

## Funcionalidades
- Cadastro de usuário
- Cadastro de eventos
- Listagem de eventos
- Participação e cancelamento de participação

## Execução
Abra o projeto em uma IDE (Eclipse, IntelliJ, Replit) e execute a classe `Main.java`.
